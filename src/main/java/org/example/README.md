Утилита предназначена для чтения текстовых файлов, классификации каждой строки по типу данных (целое число, вещественное число, строка), записи отфильтрованных данных в отдельные выходные файлы и сбора подробной статистики по каждому типу.
Проект использует Apache Maven для управления зависимостями и сборки.

Параметр	Значение
Система сборки	Apache Maven
Версия	3.x и выше

Проект полностью совместим со стандартными инструментами JetBrains.

Параметр	Значение
IDE	GIGA IDE 2024.3 (Community Edition)

В меню Run -> Edit Configurations... настриваем параметры запуска.
В поле Program arguments вводим необходимые опции и имена входных файлов (in1.txt in2.txt).
Нажимаеме Run (зеленый треугольник).
